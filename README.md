# Customer Behavior Analysis

## Project Overview

This project explores customer shopping behavior using Python, SQL, and Power BI.

The goal was to practice the complete data analysis process, from data exploration and cleaning to business analysis and dashboard creation.

## Dataset

The dataset contains information about customer demographics, purchase amounts, product categories, review ratings, subscription status, discounts, shipping methods, and previous purchases.

## Tools Used

* Python (Pandas)
* SQL
* MySQL
* Power BI

## Project Files

* `analysis.ipynb` – data exploration and analysis using Python and Pandas
* `CustomerBehavior.sql` – SQL queries used to answer business questions
* `CustomerBehavior.pbix` – Power BI dashboard
* `customer_shopping_behavior.csv` – source dataset

## Analysis Performed

Some of the analyses included:

* Revenue generated by male and female customers
* Customers who spent above average while using discounts
* Highest-rated products
* Spending comparison between shipping methods
* Subscriber vs non-subscriber revenue analysis
* Customer segmentation based on previous purchases
* Revenue contribution by age group

## Key Skills Practiced

* Data cleaning and exploration
* SQL querying and aggregations
* Customer segmentation
* Business-oriented data analysis
* Dashboard development in Power BI
* Data storytelling and visualization

## Purpose

This project was created as a portfolio project to practice real-world data analysis techniques and demonstrate skills in Python, SQL, and Power BI.
